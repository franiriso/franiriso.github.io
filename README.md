# Fran Iriso Portfolio

A personal portfolio website showcasing my engineering background, projects, and experience.

## Features

- Dark theme with laser-inspired visual effects
- Interactive tabs for About, Resume, and Projects
- Responsive design
- Space for profile photo and personal summary
- Tech stack and languages display

## Setup

1. Clone the repository
2. Open `index.html` in a web browser or run a local server:
   ```bash
   python3 -m http.server 8000
   ```
3. Navigate to `http://localhost:8000`

## Customization

- Replace `placeholder-photo.jpg` with your actual photo
- Edit the summary, tech stack, and languages in `index.html`
- Add your projects in the Projects tab
- Modify styles in `styles.css` for further customization

## Technologies Used

- HTML5
- CSS3
- JavaScript
